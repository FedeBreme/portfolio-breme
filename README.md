# 💼 Mi Portfolio Web – Federico Breme

Este proyecto corresponde a mi **portfolio profesional**, donde presento mi perfil, experiencia, habilidades técnicas y proyectos desarrollados durante mi formación en programación.

El objetivo de este sitio es **mostrar mis capacidades técnicas, proyectos y crecimiento profesional**, funcionando como una carta de presentación para el mercado laboral IT.

---

# 🚀 Tecnologías utilizadas

El portfolio fue desarrollado utilizando tecnologías de desarrollo web fundamentales:

- **HTML5** – estructura semántica del sitio  
- **CSS3** – estilos, diseño visual y responsive design  

El sitio fue construido aplicando buenas prácticas de **maquetado web, organización de estilos y diseño responsive**.

---

# 📱 Diseño Responsive

El sitio cuenta con **diseño responsive**, permitiendo una correcta visualización en distintos dispositivos:

- 🖥 Desktop  
- 💻 Laptop  
- 📱 Celulares  
- 📲 Tablets  

Esto garantiza una buena experiencia de usuario independientemente del tamaño de pantalla.

---

# 📂 Estructura del Portfolio

El sitio se encuentra organizado en diferentes secciones para presentar la información de manera clara:

## 👨‍💻 Sobre mí
Sección de presentación personal donde describo mi perfil como desarrollador, mis intereses profesionales y mi formación en programación.

## 💼 Experiencia
Apartado donde detallo mi experiencia académica y algunos proyectos desarrollados durante mi formación.

## ⚙️ Stack Tecnológico
Listado de tecnologías y herramientas que manejo, organizadas en diferentes categorías:

- Frontend
- Backend
- Bases de datos
- Herramientas de desarrollo

## 📁 Mis Proyectos
Colección de proyectos desarrollados donde se incluye:

- descripción breve del proyecto
- enlace al repositorio en GitHub
- demo en línea cuando está disponible

## 🎓 Mis Cursos
Sección dedicada a cursos y formaciones complementarias relacionadas con programación y tecnología.

## 📬 Contacto
Panel de contacto con enlaces directos a diferentes medios de comunicación profesional:

- GitHub  
- LinkedIn  
- Email  
- WhatsApp
